# 10-restserver_respaldo

Rest configurado con Express con clases.

Es un backed básico listo para ir metiéndole funcionalidad.

1. Para instalar las dependencias:

```
npm i
```

2. Renombrar el archivo example.env a .env

3. Para ejecutar:

```
npm run start:dev
```

4. Ir a la ruta

```
http:/localhost:8080
```
