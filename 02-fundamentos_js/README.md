# 02-fundamentos_js

En este punto vamos a ver un reforzamiento de ES6 y ES7.

- let vs var
- Template literales
- Destructuración
- Funciones de flecha
- Callbacks y callbacks en cadena
- Promesas y promesas en cadena
- Async y Await

Para iniciar el proyecto:

```
npm init
```

Para instalar laa dependencias, básicamente Nodemon

```
npm i
```

Para ejecutar:

```
npm run start_01
```
